实验题目

利用云平台搭建个人博客

实验学时

2

实验日期

2023.03.14

实验目的

熟悉个人博客系统的搭建。

具体包括：

参考方案：注册Github账号，搭建Hexo环境并实现个人博客搭建，撰写实验报告。

硬件环境

联网的计算机一台

硬件环境

联网的计算机一台

实验步骤与内容

1.安装Node.js

(1)从node.js下载安装软件

(2)运行下载可执行文件

(3)在终端输入 node -v 和 npm -v 检测是否安装成功

图1.检测node.j安装

2.安装Git

(1)在官网上下载，并安装

(2)将Git与GitHub进行绑定，在终端输入以下命令：git config –global user.name “用户名” 和git config –global user.email “注册邮箱”

(3)生成ssh密钥文件，在终端输入：ssh-keygen -t rsa -C “GitHub邮箱地址”，在GitHub settings keys添加新的SSH Key

​ 图2.SSH key

3.安装Hexo

(1)在电脑合适的位置新建一个文件夹作为存放Hexo的位置

(2)在终端输入命令：npm install -g hexo-cli 进行Hexo的安装

(3)初始化博客，输入命令：hexo init

​ 图3.安装Hexo

4.启动Hexo

(1)输入 hexo s 启动服务

​ 图4.启动hexo

(2)在浏览器输入地址http://localhost:4000 可以访问博客

(3)输入命令 hexo n “Hello World!”，新建博客

​ 图5.个人博客

5.变更主题

(1)安装pug和stylus，输入命令：npm install hexo-renderer-pug hexo-renderer-stylus

(2)使用git命令获取最新的主题文件，输入命令 git clone –depth 1 https://github.com/jerryc127/hexo-theme-butterfly themes/butterfly

(3)找到_config.yml文件，修改相关代码

​ 图6修改主题

(4)执行hexo g -d部署主题

​ 图7.部署主题

6.新建仓库

(1)在GitHub创建仓库，其中仓库类型为public，仓库名为用户名.github.io (2)进入仓库主页选择settings-pages，进入网页，可以看到用户名.github.io

​ 图8.博客地址

7.在GitHub上部署

(1)安装模块，输入命令 npm install hexo-deployer-git –save

(2)配置__config.yml文件

(3)推动到GitHub远程仓库

​ 图9.推送到远程仓库

(4)在浏览器输入之前在新建仓库看到的网页，得到搭建成功的博客。

结论分析与体会

  1. 利用GitHub实现了个人博客的搭建，过程简单，方便。

  2. 由于域名申请需要额外的开销，故本实验所搭建的个人博客未进行域名的绑定

  3. 在学习Hexo的过程中，由于教程中的命令出现了中英文符号的差异，直接复制教程中的命令，导致运行错误，之后所有的命令不再直接进行复制。
